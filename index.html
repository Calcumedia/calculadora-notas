Preguntas Frecuentes

¿Para quién es esta herramienta y qué soluciona? ▼

Está diseñada para aliviar el caos del cálculo manual en tres situaciones críticas: para quienes necesitan su nota exacta para becas MEC o másteres, para estudiantes compitiendo por un destino Erasmus, y para alumnos extranjeros que necesitan traducir sus notas españolas al sistema europeo de letras de forma instantánea y sin errores.

¿Qué es la nota media ponderada? ▼

A diferencia de una media simple, aquí cada asignatura tiene un "peso" distinto según sus créditos. Nuestra calculadora multiplica cada nota por sus ECTS para darte el resultado real que exige tu universidad.

¿Cómo funcionan las equivalencias ECTS? ▼

Es el sistema de letras usado en toda Europa para facilitar la movilidad. Traducimos tu nota de 0-10 a la escala A (Excelente), B (Notable Alto), C (Notable), D (Aprobado Alto), E (Suficiente) y F (Suspenso).
